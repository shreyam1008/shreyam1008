## тЪб About

**Senior Full-Stack Engineer ┬╖ 7+ years ┬╖ Open-source obsessive**

I ship frontend, backend, infra, and the opinions that come with all three.

> **Tabs vs spaces?** Whoever pays me.

### How I think about the stack

- **Frontend** тАФ The user should feel *happy*. I obsess over 2px alignment shifts, 60fps scroll, and hover states nobody consciously notices but everyone unconsciously feels. The best UI is the one where nothing feels wrong and you can't explain why.
- **Backend** тАФ Fast as physically possible. Go on all cores. I'd go Zig or Rust if I had the temperament, but Go ships today and ships clean.
- **Database** тАФ A necessary evil I've learned to respect. I don't love writing SQL at 2 AM, but the database is always right and I am usually wrong. We have an understanding.
- **Deployment** тАФ Muddy, chaotic, and I love every second of it. I would put my house in a Docker container if the landlord allowed it. Caddy, Cloudflare, GitHub Actions тАФ if it can be automated, it will be.
- **Tooling** тАФ `bun` over `node`. `uv` over `pip`. If the tool is slower than the code it manages, the tool is the bug.
